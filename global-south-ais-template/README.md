# 🚀 Global South AI Safety Hackathon - Starter Template

This repository is pre-configured for rigorous AI Safety research using **Cursor**.

## Getting Started

1. **Use this template:** Click the green "Use this template" button on GitHub to create your own repo.

2. **Clone locally:** `git clone [your-new-repo-url]`

3. **Open in Cursor:** Open the cloned folder in the Cursor IDE.

4. **Install Dependencies:** Run `pip install -r requirements.txt`.

## How the Cursor Integration Works

This project contains a `.cursorrules` file. When you use Cursor's `Ctrl/Cmd + I` (Composer) or the Chat panel, the AI will automatically:

* Optimize for the hackathon judging criteria (Impact, Execution, Presentation).

* Enforce rigorous PyTorch standards (e.g., inline tensor shape documentation).

* Help you actively maintain the `draft_submission.md` file.

**Pro-tip:** Go to Cursor Settings > Features > Docs, and add the documentation URLs for any specific frameworks you are using (like TransformerLens or OpenAI Evals) so the agent can reference them instantly using `@Docs`.
